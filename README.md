# Koshin Media Website

Welcome to the repository for **Koshin Media**, a professional portfolio and service website for video editing services tailored for content creators and brands.

## 📝 Description

Koshin Media is a sleek, modern, and fully responsive static website designed to showcase a professional video editing portfolio. The site highlights the services offered, features a dynamic portfolio gallery with embedded videos, displays client testimonials, and provides an easy-to-use contact form for prospective clients.

## ✨ Features

- **Modern UI/UX:** Clean design using the Outfit and Space Grotesk fonts.
- **Responsive Layout:** Fully functional on all devices (mobile, tablet, desktop).
- **Infinite Scrolling Partner Logos:** A dynamic, infinite scroll animation displaying trusted brands and clients.
- **Video Portfolio:** Embedded YouTube and Vimeo videos showcasing various editing styles (Talking Head, Documentary, Reels, Ads, etc.).
- **Interactive Testimonials Slider:** A clean carousel to display client reviews and ratings.
- **Working Contact Form:** Integrated with Web3Forms for seamless client inquiries.
- **Vercel Analytics:** Pre-configured script for tracking website insights.

## 🛠️ Technologies Used

- **HTML5** for semantic structure.
- **CSS3 (Vanilla)** for styling, animations, and responsive design.
- **JavaScript (Vanilla)** for interactive elements (hamburger menu, sliders, dynamic form handling).
- **FontAwesome** for scalable vector icons.
- **Google Fonts** for typography.

## 🚀 Getting Started

This is a static website, which means it requires no complex build processes or backend setup to view locally.

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd koshin-media
   ```
3. Open `index.html` in your preferred web browser. Alternatively, use a local development server like VS Code's Live Server extension for a better development experience.
